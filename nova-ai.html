Project

Nova AI

An AI‑powered assistant that helps users draft content, summarize information, and explore ideas through a clean conversational interface.

View GitHub repo Back to projects

Overview

Nova AI is a web‑based assistant designed to feel fast and focused. Users can ask questions, generate drafts, and iterate on ideas in a single interface without juggling multiple tools.

I built Nova AI to practice working with modern AI APIs, manage conversational context, and design a UI that feels professional enough for real‑world use.

Tech stack

TypeScript, React, Node.js/Express, REST APIs, and a hosted LLM provider (e.g., OpenAI or similar), deployed on a cloud platform.

Highlights

  • Supports multi‑turn conversations with preserved context and regenerated responses.
  • Includes presets for tasks like summarization, idea generation, and email drafting.
  • Implements input validation, loading states, and clear error messages for reliability.
  • Responsive layout that works on mobile, tablet, and desktop.
