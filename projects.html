Numerical-Aided Machine Learning for Low-RCS Orbital Debris Tracking

This work studies how to detect and track low radar-cross-section (Low-RCS) orbital debris in LEO using a numerical-aided, evaluation-driven ML pipeline that can run under tight real-time and hardware constraints.

The key idea is to treat ML as a component inside a full sensing system. We reduce the search space using orbital propagation priors, then run a compact neural detector on constrained hardware, and finally fuse detections across nodes to improve reliability.

Focus

  • Reliability: stable behavior under weak returns, clutter, and varying aspect angles.
  • Constraints: predictable latency and memory limits in an embedded / FPGA-friendly design.
  • Evaluation: detection performance under controlled false-alarm rates, plus ablations.
