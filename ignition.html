Ignition

A starter kit that bundles core pieces like auth, routing, and basic UI so new projects can ship a useful prototype quickly.

Overview

Ignition is meant to remove the boring setup work from new ideas. Instead of wiring auth, routing, and layout from scratch every time, it gives you a clean baseline.
