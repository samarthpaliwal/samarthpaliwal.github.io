Numerical-Aided Machine Learning for Low-RCS Orbital Debris Tracking

This work studies how to detect and track low radar-cross-section (Low-RCS) orbital debris in LEO using a numerical-aided, evaluation-driven ML pipeline that can run under real-time and hardware constraints.

The key idea is to treat ML as one component inside a sensing system. We reduce the search space using orbital propagation priors, run a compact detector on constrained hardware, and fuse detections across nodes.

Focus

  • Reliability: stable behavior under weak returns, clutter, and varying aspect angles.
  • Constraints: predictable latency and memory limits in an embedded / FPGA-friendly design.
  • Evaluation: detection under controlled false-alarm rates, plus ablations.
