Numerical-Aided Machine Learning for Low-RCS Orbital Debris Tracking

This research investigates how machine learning, signal processing, and FPGA acceleration can be combined to reliably detect and track low radar-cross-section (Low-RCS) orbital debris in Low Earth Orbit (LEO).

The work addresses a real operational problem: traditional radar pipelines struggle with weak returns, latency constraints, and scalability when tracking small debris objects. We propose a system that integrates catalog-aware gating, FPGA-resident neural inference, and multistatic fusion to improve detection reliability while staying within tight resource budgets.

Key contributions

  • Ephemeris-constrained gating using TLE/SATNOGS priors to reduce search space.
  • Quantized 1D-CNN inference deployed directly on FPGA for real-time throughput.
  • Multistatic radar fusion to increase aspect diversity and resolution.
  • Latency-aware system design with explicit DSP, BRAM, and timing constraints.
  • MPC-supervised closed-loop control for adaptive gating and thresholds.

System architecture

Candidate debris trajectories are propagated using SGP4/SDP4 models to generate time-tagged predictions. These predictions define a timeâ€“rangeâ€“Dopplerâ€“angle gating window that constrains downstream processing.

Within this window, raw I/Q radar data is converted into compact slow-time / fast-time tensors. A quantized neural network executes fully on FPGA, producing detection logits while meeting real-time latency constraints. A single-board computer (SBC) handles scheduling, model loading, and telemetry.

Evaluation and validation

  • Probability of detection measured at fixed false-alarm rates.
  • Latency breakdown across acquisition, gating, inference, and fusion.
  • Ablation studies isolating gating, fusion, and quantization effects.
  • Theoretical analysis of gating volume vs. miss probability.

This work demonstrates how ML systems can be designed as engineering artifacts, not black boxes â€” respecting physics, hardware limits, and operational reliability. The ideas extend beyond radar to any real-time ML system operating under uncertainty.
